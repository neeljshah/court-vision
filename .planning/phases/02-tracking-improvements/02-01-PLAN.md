  <action>
    Create `src/tracking/jersey_ocr.py` implementing the full OCR and color-clustering pipeline exactly as specified in the research doc (Pattern 1 and Pattern 4). The file must stay under 300 lines.

    Module constants at top:
    ```python
    _OCR_CONF_MIN = 0.65   # minimum EasyOCR confidence to accept a digit read
    _MIN_CROP_PIXELS = 600 # below this, fall back from k-means to mean color
    _KMEANS_K = 3          # number of clusters for jersey color
    ```

    Functions to implement (all need docstrings + type hints):

    1. `get_reader() -> easyocr.Reader`
       - Lazy-init singleton. `easyocr.Reader(['en'], gpu=True, verbose=False)`
       - Store in module-level `_reader: Optional[easyocr.Reader] = None`

    2. `preprocess_crop(crop_bgr: np.ndarray) -> np.ndarray`
       - Slice rows 20%-70% of crop height (jersey area)
       - If slice is empty, return `np.zeros((64, 32), dtype=np.uint8)`
       - Upscale to at least 64px tall with `cv2.INTER_CUBIC`
       - Convert to grayscale, apply CLAHE `clipLimit=3.0, tileGridSize=(4,4)`
       - Apply adaptive threshold: `cv2.adaptiveThreshold(enhanced, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)`
       - Return the binary image (helps dark jerseys — see research Pitfall 1)

    3. `read_jersey_number(crop_bgr: np.ndarray) -> Optional[int]`
       - Call `preprocess_crop`, then `get_reader().readtext(img, allowlist='0123456789', detail=1, paragraph=False, width_ths=0.7, min_size=5)`
       - Also run OCR on `cv2.bitwise_not(preprocessed)` (inverted) and take the higher-confidence result
       - Accept first digit result with conf >= `_OCR_CONF_MIN` where `text.isdigit()` and `0 <= int(text) <= 99`
       - Return `int` or `None`. Wrap entire body in try/except returning `None` on any exception.

    4. `dominant_hsv_cluster(crop_bgr: np.ndarray, k: int = _KMEANS_K) -> np.ndarray`
       - Slice upper 70% (jersey, not shorts)
       - If `roi.size < _MIN_CROP_PIXELS`, return `cv2.cvtColor(roi.reshape(1,-1,3), cv2.COLOR_BGR2HSV).reshape(-1,3).mean(axis=0).astype(np.float32)` (mean fallback)
       - Otherwise: convert to HSV, reshape to (-1, 3), run `KMeans(n_clusters=k, n_init=3, max_iter=30, random_state=0)`, return centroid of largest cluster as float32 shape (3,)
       - Import: `from sklearn.cluster import KMeans`
  </action>
