## What Was Decided This Session

### Full Project Vision Locked In
- End goal: world's best NBA analytics + prediction system
- Three products: Betting Dashboard, Analytics Dashboard, AI Chat
- See [[Project Vision]] for full detail

### Dataset Strategy
- Every game tracked through the pipeline builds the dataset automatically
- CV tracking (anonymous positions + context) + NBA API enrichment (outcomes + labels) = joined ML dataset
- 20 games → shot quality model, 50 → possession model, 200+ → live win probability LSTM

### ML Model Build Order
1. Pre-game win probability (XGBoost, pure NBA API — build first, no tracking needed)
2. Player prop models (points, rebounds, assists)
3. Shot quality + possession outcome (need tracking volume)
4. Live win probability LSTM (needs 200+ full games)
5. Lineup chemistry model

### Betting Integration
- The Odds API for live sportsbook lines
- Model prediction vs implied odds → edge detection
- Player props are least efficient market = most edge potential

### Tech Stack Decided
- Backend: FastAPI (Python)
- Frontend: React + Next.js + D3
- AI Chat: Claude API with tool use (calls backend endpoints for real data)
- DB: PostgreSQL
